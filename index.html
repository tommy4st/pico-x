A minimal component and utility layer on top of Pico CSS v2. Semantic HTML first, class-light, fully themeable.

Variant System

Add variant="name" to any element to remap all --pico-primary-* tokens. Works on buttons, cards, tags, callouts. You can also set --color directly via inline style.

Size attribute

Small text · Normal text · Large text · X-Large text 

<!-- Semantic variant -->
<button variant="success">Save</button>

<!-- Named color -->
<button variant="indigo">Indigo</button>

<!-- Custom inline color -->
<button style="--color: #e91e63">Custom</button>

<!-- Variant on any element -->
<article variant="info">
  <h4>Info Card</h4>
  <p>Content here.</p>
</article>

<!-- Size attribute -->
<span size="small">Small</span>
<span size="large">Large</span>

Callouts

Contextual message blocks using .callout with optional variant. Pure CSS, no JavaScript needed.

Multi-select

Dropdown with tag-based selection via <pico-multiselect>. Define options as <option> children.

Selected: TypeScript, Rust

<pico-multiselect name="languages" placeholder="Select…">
  <option value="js">JavaScript</option>
  <option value="ts" selected>TypeScript</option>
  <option value="py">Python</option>
  <option value="rs" selected>Rust</option>
</pico-multiselect>

Toasts

Temporary notifications via PicoToast.show(message, options). Styled as callouts, positioned top-right. Auto-dismiss after 3s; set duration: 0 for persistent. 

import { PicoToast } from "./js/pico.js";

PicoToast.show("Saved!", { variant: "success" });
PicoToast.show("Oops!", { variant: "danger", duration: 5000 });
PicoToast.show("Sticky", { variant: "info", duration: 0 });

// Dismiss programmatically
const t = PicoToast.show("Working...");
PicoToast.dismiss(t);

Modal

Uses the native <dialog> element with Pico CSS transitions. Open/close via PicoModal. 

<button data-target="my-modal" class="modal-trigger">Open</button>

<dialog id="my-modal">
  <article>
    <header>
      <button aria-label="Close" rel="prev" class="modal-close"></button>
      <h3>Modal Title</h3>
    </header>
    <p>Content goes here.</p>
    <footer>
      <button class="secondary modal-close">Cancel</button>
      <button class="modal-close">Confirm</button>
    </footer>
  </article>
</dialog>

<script type="module">
  import { PicoModal } from "./js/pico.js";
  PicoModal.open(document.getElementById("my-modal"));
  PicoModal.close(document.getElementById("my-modal"));
</script>

Usage

Include the CSS and import the JS module. All web components auto-register on import. 

<link rel="stylesheet" href="pico.css">

<script type="module">
  import {
    PicoTag, PicoRating, PicoMultiselect,
    PicoToast, PicoModal, Pico
  } from "./js/pico.js";

  PicoToast.show("Hello!", { variant: "success" });
  PicoModal.open(document.getElementById("my-dialog"));
</script>
